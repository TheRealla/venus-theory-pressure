## Concept

Venus Theory – Pressure is a battery-powered, tubular keytar-style bass instrument that combines:
- Primitive 1960s organ-bass soul (Tubon DNA)
- Modern bass machine power (Braumah-style punch and glide)
- Mythological seduction (Venus gluck/vocal tube resonance)
- Real Beatles bass intelligence (two dedicated "Beatles Secrets" themes)

No menus. No polyphony. No filter sweeps. Just twist, swell, and play.

## Core Features

- **Strictly monophonic** — last-note priority with 20–40 ms legato glide
- **100 hard-switched presets** — all "mastered out of the box"
- **BreakTweaker-style transient shaping** — attack bloom + sustain sculpt
- **Dual compression**: Aggressive NY parallel + smooth optical Sculpt
- **Gluck / vocal tube resonance** — drive-dependent formant character
- **Battery sag emulation** — natural pumping and darkening
- **Global irregular vibrato** (4–8 Hz with jitter)
- **Manual swell slider** under right-hand grip + expression pedal input
- **Two selectable Beatles Secrets themes** on the mechanical rotary:
  - **Melodic Weaver** — warm, singing, chord-tone weaving (Paul-style melodic invention)
  - **Driving Pulse** — rhythmic, pocket-perfect drive with heavier pump

## Hardware
- 75–80 cm tubular black chassis
- 32 mini-keys (bass-centric range E0–E3)
- Mechanical rotary selector with detents
- Right-hand swell slider + optional downward pitch rocker
- Strap buttons at both ends
- Mono ¼" output + internal speaker mode
